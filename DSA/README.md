# NED-Software-Engineering-Department-Labs

This repository is a comprehensive archive of my laboratory work and academic projects. It tracks my progression from basic programming logic to complex data structures, database management, and professional software construction.

## 📚 Subject Directories

### 🏗️ Data Structures & Algorithms (DSA)
*Focus: Problem-solving and computational efficiency.*
* **Implementation:** Custom Linked Lists, Stacks, Queues, Trees, and Graphs.
* **Algorithms:** Sorting (Quick, Merge, Heap), Searching, and Pathfinding.
* **Analysis:** Evaluating algorithms based on Big O notation for time and space complexity.

### 🗄️ Database Management Systems (DBMS)
*Focus: Data integrity and relational modeling.*
* **SQL:** Complex queries, sub-queries, joins, and indexing.
* **Architecture:** ER Diagram design and Database Normalization (1NF through BCNF).
* **Integration:** Connecting backend systems (Node.js) with Oracle Database environments.

### 🏛️ Object-Oriented Programming (OOPS)
*Focus: Reusable and modular software design.*
* **Core Concepts:** Encapsulation, Inheritance, Polymorphism, and Abstraction.
* **Applications:** Real-world system modeling using Classes and Objects to ensure code maintainability.

### 🛠️ Software Construction & Development (SCD)
*Focus: Professional development workflows.*
* **Content:** Design patterns, Unit Testing, and Version Control.
* **Practices:** Writing clean, documented, and modular code ready for production environments.

### ⌨️ Programming Fundamentals (PF)
*Focus: Procedural logic and syntax mastery.*
* **Basics:** Control structures (Loops, If-Else), Arrays, Pointers, and Memory Management.
